# BeyondChats_Laravel_CRUD_Full_Project_FINAL
This project is a full-stack Laravel-based CRUD application that scrapes and manages blog articles from the BeyondChats website. The backend is built using Laravel REST APIs, while the frontend is implemented using a simple HTML and JavaScript interface.  The application stores the five oldest blog articles from the BeyondChats blogs section 
Live Working link:-https://beyondchatslaravelcurdfullproject.lovable.app/
